Make a Request

Send a POST request with your message context.

request.js 
import fetch from 'node-fetch';

const response = await fetch('https://pakbot.api/chat', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'pk_your_api_key_here'
  },
  body: JSON.stringify({ 
    message: 'Pakistan ki history kya hai?',
    sessionId: 'user-123'
  })
});

const data = await response.json();
console.log(data.reply);

API Endpoints

Complete REST coverage for advanced use cases.

POST /api/chat

Send a message and receive an AI reply. Supports optional image (base64) and url fields.

POST /api/chat/stream

Same as /chat but streams the reply token-by-token via Server-Sent Events.

GET /api/bots

List all registered developer bots

POST /api/bots

Create a new custom bot profile

GET /api/bots/:id/conversations

List conversations for a bot

POST /api/bots/:id/conversations

Start a new conversation session

POST /api/bots/:id/webhooks

Register a webhook for events
